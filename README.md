Projeto em Desenvolvimento: ERP para Vendas de Lanches – ASP.NET Core MVC

Atualmente estou desenvolvendo um sistema web estilo ERP voltado para vendas de lanches, utilizando ASP.NET Core MVC e Entity Framework Core. O objetivo do projeto é criar uma aplicação funcional e moderna, aplicando os principais conceitos de desenvolvimento web com foco em arquitetura limpa e boas práticas.

Funcionalidades e recursos em desenvolvimento:
	•	Estrutura completa baseada no padrão MVC;
	•	Mapeamento de entidades com Entity Framework Core (abordagem Code First);
	•	Validação com Data Annotations;
	•	Implementação dos padrões Repository e ViewModel;
	•	Controle de sessão com carrinho de compras;
	•	Autenticação e autorização com ASP.NET Core Identity (login, logout e registro de usuários);
	•	Uso de View Components e Partial Views;
	•	Paginação e filtros de dados;
	•	Relatórios com consultas LINQ e exportação em PDF com FastReport Open Source;
	•	Geração de gráficos interativos com Google Charts.

Este projeto está sendo fundamental para aprofundar meus conhecimentos em desenvolvimento web com .NET e consolidar minha experiência prática com tecnologias modernas da Microsoft.
