🍔 ERP de Vendas de Lanches – ASP.NET Core MVC
Sistema web em desenvolvimento com foco na criação de um ERP completo para gerenciamento de vendas de lanches, utilizando ASP.NET Core MVC e arquitetura limpa. A solução visa unir produtividade, escalabilidade e boas práticas de desenvolvimento em um único projeto.

🚀 Objetivo
Criar uma aplicação robusta e extensível, com recursos como:

Gerenciamento de pedidos e produtos

Controle de usuários e permissões

Relatórios e dashboards com dados em tempo real

Tudo seguindo os pilares da arquitetura MVC, SOLID e Clean Code.

🔧 Tecnologias
ASP.NET Core MVC

Entity Framework Core (Code First)

ASP.NET Core Identity

LINQ

FastReport Open Source

Google Charts

Razor Views, View Components e Partial Views

🧩 Funcionalidades (em construção)
Estrutura modular com separação de responsabilidades (MVC + ViewModel)

Autenticação e autorização com Identity (registro, login, logout)

Carrinho de compras com controle de sessão

CRUD completo com validação via Data Annotations

Paginação, ordenação e filtros com LINQ

Relatórios em PDF com FastReport

Gráficos dinâmicos com Google Charts

Repositório genérico e injeção de dependência

📌 Status
🛠️ Em desenvolvimento
Base sólida pronta, com foco agora em expandir funcionalidades, testes e integrações avançadas.
