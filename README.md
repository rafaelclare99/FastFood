🍔 ERP de Vendas de Lanches - ASP.NET Core MVC

Este projeto está em desenvolvimento e tem como objetivo criar um sistema web completo (ERP) para gerenciamento de vendas de lanches, utilizando tecnologias modernas da Microsoft como ASP.NET Core MVC e Entity Framework Core.

🚀 Objetivo

Construir uma aplicação funcional com recursos de gerenciamento de pedidos, controle de usuários, relatórios e visualização de dados, seguindo boas práticas de desenvolvimento e arquitetura limpa.

🔧 Tecnologias Utilizadas
	•	ASP.NET Core MVC
	•	Entity Framework Core (Code First)
	•	ASP.NET Core Identity
	•	LINQ
	•	FastReport Open Source
	•	Google Charts
	•	Razor Views, View Components e Partial Views

🧩 Funcionalidades em Desenvolvimento
	•	Estrutura baseada em arquitetura MVC
	•	Mapeamento de entidades e migração de banco de dados (Code First)
	•	Validação de dados com Data Annotations
	•	Padrão Repository e uso de ViewModel
	•	Carrinho de compras com controle de sessão
	•	Sistema de autenticação e autorização com Identity (Login, Logout e Registro)
	•	Paginação e filtros de dados
	•	Relatórios gerados com LINQ e exportação em PDF (FastReport)
	•	Criação de gráficos com Google Charts

📌 Status

🔨 Em desenvolvimento
Este projeto está em andamento e serve como base prática para aplicação de conceitos avançados em ASP.NET Core.

📚 Aprendizados

	•	Criação de sistemas seguros com autenticação e controle de acesso
